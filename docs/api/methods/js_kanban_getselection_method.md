# getSelection()

### Description

Gets an array of IDs of the selected cards

### Usage

`getSelection: () => array;`


### Returns

The method returns an array of IDs of the selected cards

### Example

```jsx
// get an array of IDs of the selected cards
kanban.getSelection();
```
