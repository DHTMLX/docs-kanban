# TypeScript 支持

从 v1.1 版本开始，DHTMLX Kanban 内置了 TypeScript 类型定义。

:::note
请在 [Snippet Tool](https://snippet.dhtmlx.com/2blo6hx8?tag=kanban) 中体验 TypeScript 示例。
:::

## TypeScript 的优势

TypeScript 通过静态类型检查和自动补全功能，能够在早期发现错误。内置的类型定义还会在您调用 Kanban API 时显示预期的数据结构。
