# Поддержка TypeScript

Начиная с версии 1.1, DHTMLX Kanban поставляется с определениями TypeScript, поэтому его можно сразу использовать с TypeScript.

:::note
Вы можете опробовать всё в <a href="https://snippet.dhtmlx.com/2blo6hx8?tag=kanban" target="_blank">Snippet Tool</a>.
:::

## Преимущества использования TypeScript

Использование DHTMLX Kanban с TypeScript даёт ряд удобных преимуществ.

TypeScript помогает сделать процесс разработки более удобным и надёжным. Благодаря проверке типов и автодополнению ошибки можно быстрее выявлять на ранних этапах. Кроме того, при работе с API DHTMLX Kanban TypeScript предоставляет понятные подсказки о том, какие типы данных использовать.
