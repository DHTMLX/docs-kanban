# DELETE `/cards/{cardId}/comments/{commentId}`

### Описание

@short: Удаляет комментарий из карточки

Этот маршрут обрабатывает **HTTP DELETE** запрос, отправленный на эндпоинт `cards/{cardId}/comments/{commentId}`.

### Параметры пути

В URL запроса включены следующие параметры:

| Название    | Тип         | Описание |
| ----------- | ----------- | ----------- |
| `cardId`    |  number     | *Обязательный*. Идентификатор карточки, из которой будет удалён комментарий. |
| `commentId` |  number     | *Обязательный*. Идентификатор комментария, который необходимо удалить. |

### Тело запроса

Отправлять тело запроса не требуется.

### Ответ

Сервер возвращает JSON-объект с ID пользователя, который удалил комментарий.

Пример:

~~~json
{
    "id": 4
}
~~~

Код HTTP-статуса указывает результат: 200 означает успешное выполнение запроса, а 500 - ошибку.
