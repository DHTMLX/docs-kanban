# DELETE `/rows`

### Описание

@short: Удаляет данные в строке (swimlane)

Маршрут обрабатывает **HTTP DELETE** запрос, отправленный на путь `/rows/{id}`.

### Параметры пути

Следующий параметр передается в строке запроса:

| Имя        | Тип         | Описание |
| ---------- | ----------- | -------- |
| `id`       | number      | *Обязательный*. ID строки, которую нужно удалить.|

### Тело запроса

Тело запроса не требуется.

### Ответ

Данные в ответе не возвращаются. HTTP статус код показывает, успешно ли выполнен запрос (response.status == 200) или произошла ошибка (response.status == 500).
