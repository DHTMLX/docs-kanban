# DELETE `/rows`

### Описание

@short: Удаляет строку (swimlane) из данных.

Этот маршрут обрабатывает **HTTP DELETE** запрос, отправленный на конечную точку `/rows/{id}`.

### Параметры пути

В URL запроса включён следующий параметр:

| Название   | Тип         | Описание |
| ---------- | ----------- | -------- |
| `id`       |  number     | *Обязательно*. Уникальный идентификатор строки, которую нужно удалить.|

### Тело запроса

Отправлять тело запроса не требуется.

### Ответ

Содержимое не возвращается. HTTP статус-код указывает результат: статус 200 означает успешное удаление, а 500 - ошибку.
