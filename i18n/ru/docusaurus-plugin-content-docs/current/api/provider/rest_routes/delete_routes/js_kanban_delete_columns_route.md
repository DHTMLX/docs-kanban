# DELETE `/columns`

### Описание

@short: Удаляет данные колонки

Этот маршрут обрабатывает **HTTP DELETE** запросы, отправленные на конечную точку `/columns/{id}`.

### Параметры пути

В URL запроса включён следующий параметр:

| Название  | Тип         | Описание                                |
| --------- | ----------- | ------------------------------------- |
| `id`      | number      | *Обязательный*. Указывает ID колонки для удаления. |

### Тело запроса

Тело запроса не требуется.

### Ответ

Контент не возвращается. Код статуса указывает результат: успех при response.status == 200 или ошибка при response.status == 500.
