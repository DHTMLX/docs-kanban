# DELETE `/columns`

### Описание

@short: Удаляет данные по колонке

Маршрут обрабатывает **HTTP DELETE** запрос, отправленный на путь `/columns/{id}`.

### Параметры пути

В строке запроса передается следующий параметр:

| Название  | Тип         | Описание                     |
| --------- | ----------- | ----------------------------|
| `id`      | number      | *Обязательный*. ID колонки, которую нужно удалить.|

### Тело запроса

Тело запроса не требуется.

### Ответ

Данные в ответе не возвращаются. HTTP статус код показывает, успешно ли выполнен запрос (response.status == 200) или произошла ошибка (response.status == 500).
