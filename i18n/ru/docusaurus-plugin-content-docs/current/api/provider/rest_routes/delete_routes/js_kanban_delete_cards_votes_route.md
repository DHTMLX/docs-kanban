# DELETE `/cards/{cardId}/vote`

### Описание

@short: Удаляет голос с карточки и возвращает json объект с ID пользователя, который удалил голос.

Маршрут обрабатывает **HTTP DELETE** запрос, отправленный на путь `/cards/{cardId}/vote`.

### Параметры пути

В строке запроса передается следующий параметр:

| Название  | Тип         | Описание                                               |
| --------- | ----------- | ------------------------------------------------------ |
| `cardId`  | number      | *Обязательный*. ID карточки, с которой будет удалён голос.|

### Тело запроса

Тело запроса не требуется.

### Ответ

Маршрут возвращает json объект с ID пользователя, который удалил голос.

Пример:

~~~json
{ 
   "id": 1
}
~~~

HTTP статус код показывает, успешен ли запрос (response.status == 200) или произошла ошибка (response.status == 500).
