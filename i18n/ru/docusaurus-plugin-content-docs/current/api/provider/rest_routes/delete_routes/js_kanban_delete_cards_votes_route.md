# DELETE `/cards/{cardId}/vote`

### Описание

@short: Удаляет голос с карточки и возвращает JSON объект с ID пользователя, который удалил голос

Этот маршрут обрабатывает **HTTP DELETE** запрос, отправленный на `/cards/{cardId}/vote`.

### Параметры пути

В URL запроса включён следующий параметр:

| Имя        | Тип         | Описание |
| ---------- | ----------- | ----------- |
| `cardId`   |  number     | *Обязательный*. ID карточки, с которой будет удалён голос.|

### Тело запроса

Тело запроса не требуется.

### Ответ

Ответ - это JSON объект, содержащий ID пользователя, который удалил свой голос.

Пример:

~~~json
{ 
   "id": 1
}
~~~

HTTP статус код указывает, был ли запрос успешным (response.status == 200) или произошла ошибка (response.status == 500).
