# DELETE `/cards`

### Описание

@short: Удаляет данные на карточке

Маршрут обрабатывает **HTTP DELETE** запрос, отправленный по пути `/cards/{id}`.

### Параметры пути

В строке запроса передается следующий параметр:

| Имя        | Тип         | Описание   |
| ---------- | ----------- | ---------- |
| `id`       |  number     | *Обязательный*. ID карточки, которую нужно удалить.|

### Тело запроса

Тело запроса не требуется.

### Ответ

Данные не возвращаются. HTTP статус код показывает, успешен ли запрос (response.status == 200) или произошла ошибка (response.status == 500).
