# GET `/uploads`

### Описание

@short: Получает запрошенный бинарный файл с сервера

Маршрут обрабатывает **HTTP GET** запрос, сделанный по пути `/uploads/{id}/{name}`.

### Параметры пути

В строке запроса передаются следующие параметры:

| Имя        | Тип         | Описание |
| ---------- | ----------- | -------- |
| `id`       |  number     | *Обязательно*. ID требуемого файла. |
| `name`     |  string     | *Обязательно*. Имя запрашиваемого файла. |

### Тело запроса

Тело запроса не требуется.

### Ответ

Маршрут возвращает запрошенный бинарный файл.

HTTP статус код указывает, успешен ли запрос (response.status == 200) или произошла ошибка (response.status == 500).
