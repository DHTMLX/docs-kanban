# POST `/cards/{cardId}/vote`

### Описание

@short: Добавляет новый голос к карточке и возвращает json-объект с ID пользователя, проголосовавшего

Этот маршрут обрабатывает **HTTP POST** запрос, отправленный по пути `/cards/{cardId}/vote`.

### Тело запроса

Тело запроса не требуется.

### Ответ

Маршрут возвращает json-объект с ID пользователя, который проголосовал.

Пример:

~~~json
{ 
   "id": 1
}
~~~

HTTP-статус код показывает, был ли запрос успешным (response.status == 200) или произошла ошибка (response.status == 500).
