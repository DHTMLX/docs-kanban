# POST `/cards/{cardId}/vote`

### Описание

@short: Добавляет новый голос к карточке и возвращает JSON-объект с ID пользователя, который проголосовал.

Этот маршрут принимает **HTTP POST** запрос по адресу `/cards/{cardId}/vote`.

### Тело запроса

Отправлять тело запроса не требуется.

### Ответ

В ответе возвращается JSON-объект с ID пользователя, который проголосовал.

Пример:

~~~json
{ 
   "id": 1
}
~~~

HTTP-статус код указывает на результат: статус 200 означает успешное выполнение запроса, а 500 - ошибку.
