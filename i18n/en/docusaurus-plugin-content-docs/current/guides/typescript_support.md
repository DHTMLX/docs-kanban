# TypeScript support

From version 1.1, DHTMLX Kanban comes with TypeScript definitions included. TypeScript works right out of the box with the library.

:::note
The <a href="https://snippet.dhtmlx.com/2blo6hx8?tag=kanban" target="_blank">Snippet Tool</a> is a great place to check out how everything works.
:::

## Advantages of using TypeScript

So, what's good about using DHTMLX Kanban with TypeScript?

TypeScript helps make development smoother and more efficient. With type checking and autocomplete, catching mistakes early becomes much easier. Plus, TypeScript gives clear hints about what kind of data to use with the DHTMLX Kanban API, which helps keep things on track while building apps.
