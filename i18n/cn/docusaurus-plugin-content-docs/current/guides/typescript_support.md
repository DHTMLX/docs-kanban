# TypeScript 支持

从 1.1 版本开始，DHTMLX Kanban 自带 TypeScript 类型定义，因此可以立即与 TypeScript 搭配使用。

:::note
欢迎在 <a href="https://snippet.dhtmlx.com/2blo6hx8?tag=kanban" target="_blank">Snippet Tool</a> 中进行试用和探索。
:::

## 使用 TypeScript 的优势

将 DHTMLX Kanban 与 TypeScript 结合使用带来诸多便捷优势。

TypeScript 能让开发过程更加顺畅和可靠。通过类型检查和自动补全，可以更容易及早发现错误。此外，在使用 DHTMLX Kanban API 时，TypeScript 还能为所需数据类型提供清晰提示。
