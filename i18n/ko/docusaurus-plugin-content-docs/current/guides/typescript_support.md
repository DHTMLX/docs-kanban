:::note
TypeScript 예제는 [Snippet Tool](https://snippet.dhtmlx.com/2blo6hx8?tag=kanban)에서 바로 체험해 볼 수 있습니다.

## TypeScript의 장점

TypeScript는 정적 타입 확인과 자동 완성 기능을 통해 실수를 조기에 발견할 수 있습니다. 내장된 정의는 Kanban API를 호출할 때 필요한 데이터 형식도 알려줍니다.
